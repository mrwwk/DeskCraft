Ops Signal Board

Build a real, previewable dashboard from local JSON data. The app must be served locally and verified in Chrome at http://127.0.0.1:8125/index.html.

Required UI system

  1. Create index.html, styles.css, app.js, and manifest.json inside the project folder.
  2. Load data.json with fetch() and render a KPI grid into #kpi-grid.
  3. Create a filter bar #filter-bar with buttons for All, Active, and Risk.
  4. Render a timeline into #incident-timeline, a canvas chart into canvas#chartCanvas, and a side drawer as aside#detail-drawer.
  5. Use the local asset assets/ops-badge.svg in the header or hero area.
  6. Manifest must set project=ops-signal-board, theme=midnight-signal, chartType=bar, dataSource=data.json, and previewPort=8125.
  7. Finish with the live preview open in Chrome.
