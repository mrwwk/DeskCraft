Launch Story Lab

Build a richer generative UI page from local structured content, then verify it in Chrome at http://127.0.0.1:8126/index.html.

Required experience

  1. Create a storytelling page with main#story-app, section#hero, section#insight-grid, section#milestones, canvas#sentimentChart, and dialog#noteDialog.
  2. Load story.json and use it to render 4 insight cards and 3 milestone items.
  3. Use the local asset assets/launch-mark.svg in the hero or header area.
  4. Implement an interaction that opens the dialog or toggles a detail panel when a milestone button is clicked.
  5. Draw a bar or column chart for the sentiment series on the canvas.
  6. Manifest must set project=launch-story-lab, theme=orchid-night, chartType=bar, dataSource=story.json, and previewPort=8126.
